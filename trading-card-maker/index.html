Trading Card Creator

Load a local JSON file with VeinBreaker abilities. Example schema:

{
  "abilities": [
    {
      "path": "spellforged",
      "tier": 3,
      "name": "Law of the Stormcore",
      "type": "attack",
      "cost": 3,
      "cooldown": 3,
      "effect": "1d8 + INT lightning; on hit, chain a free 1d4 bolt to another target; if Momentum >= 2, bolt deals 1d6."
    }
  ]
}

Cost and cooldown render inside the art window. Tags, images, and other extra fields are ignored unless present.

TradingCard JSON schema:

{
  "cards": [
    {
      "name": "Law of the Stormcore",
      "type": "Attack",
      "attack": 8,
      "defense": 3,
      "description": "1d8 + INT lightning; on hit chain a free 1d4 bolt to a second target; if Momentum \u2265 2, that bolt deals 1d6 instead",
      "imageUrl": "images/stormcore.jpg",
      "rarity": "rare",
      "setSymbol": "VB"
    }
  ]
}

The imageUrl can be a relative path next to this page or a data URL (e.g. data:image/png;base64,...).
